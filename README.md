
# Chart GPT

A platorm for generate chart with ChatGPT

## Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- ChatGPT 3.5

## Installation

Install ChartGPT with Yarn

```bash
  cd ChartGPT
  yarn
  yarn start
```
